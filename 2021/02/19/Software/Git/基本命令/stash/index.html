git stash

本文最后更新于：2021年2月19日 晚上

git stash

  • 保存当前工作进度,会把暂存区和工作区的改动保存起来

list

  • 显示保存进度的列表,也就意味着,git stash命令可以多次执行

pop

  • 通过git stash pop命令恢复进度后,会删除当前进度

apply

  • 除了不删除恢复的进度之外,其余和git stash pop 命令一样

drop

  • 删除一个存储的进度,如果不指定stash_id,则默认删除最新的存储进度

clear

  • 删除所有存储的进度
