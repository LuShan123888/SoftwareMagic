SQL еөҢеҘ—жҹҘиҜў

  • еңЁSQLиҜӯиЁҖдёӯ,дёҖдёӘSELECT-FROM-WHEREиҜӯеҸҘз§°дёәдёҖдёӘжҹҘиҜўеқ—,е°ҶuigeжҹҘиҜўеқ—еөҢеҘ—еңЁеҸҰдёҖдёӘжҹҘиҜўеқ—зҡ„WHEREеӯҗеҸҘжҲ–HAVINGзҹӯиҜӯзҡ„жқЎд»¶дёӯзҡ„жҹҘиҜўз§°дёәеөҢеҘ—жҹҘиҜў(nested query)дҫӢеҰӮ:

  • еңЁжң¬дҫӢдёӯ,дёӢеұӮжҹҘиҜўеқ—SELECT Sno FROM SC WHERE Cno='2вҖҷжҳҜеөҢеҘ—еңЁдёҠеұӮжҹҘиҜўеқ—SELECT Sname FROM Student WHERE Sno INзҡ„WHEREжқЎд»¶дёӯзҡ„,дёҠеұӮзҡ„жҹҘиҜўеқ—з§°дёәеӨ–еұӮжҹҘиҜўжҲ–зҲ¶жҹҘиҜў,дёӢеұӮжҹҘиҜўеқ—з§°дёәеҶ…еұӮжҹҘиҜўжҲ–еӯҗжҹҘиҜў

  • SQLиҜӯиЁҖе…Ғи®ёеӨҡеұӮеөҢеҘ—жҹҘиҜў,еҚідёҖдёӘеӯҗжҹҘиҜўдёӯиҝҳеҸҜд»ҘеөҢеҘ—е…¶д»–еӯҗжҹҘиҜў,йңҖиҰҒзү№еҲ«жҢҮеҮәзҡ„жҳҜ,еӯҗжҹҘиҜўзҡ„SELECTиҜӯеҸҘдёӯдёҚиғҪдҪҝз”ЁORDER BYеӯҗеҸҘ,ORDER BYеӯҗеҸҘеҸӘиғҪеҜ№жңҖз»ҲжҹҘиҜўз»“жһңжҺ’еәҸ

  • еөҢеҘ—жҹҘиҜўдҪҝз”ЁжҲ·еҸҜд»Ҙз”ЁеӨҡдёӘз®ҖеҚ•жҹҘиҜўжһ„жҲҗеӨҚжқӮзҡ„жҹҘиҜў,д»ҺиҖҢеўһејәSQLзҡ„жҹҘиҜўиғҪеҠӣ,д»ҘеұӮеұӮеөҢеҘ—зҡ„ж–№ејҸжқҘжһ„йҖ зЁӢеәҸжӯЈжҳҜSQLдёӯ"з»“жһ„еҢ–"зҡ„еҗ«д№үжүҖеңЁ

еёҰжңүINи°“иҜҚзҡ„еӯҗжҹҘиҜў

еңЁеөҢеҘ—жҹҘиҜўдёӯеӯҗжҹҘиҜўзҡ„з»“жһңеҫҖеҫҖжҳҜдёҖдёӘйӣҶеҗҲ,жүҖд»Ҙи°“иҜҚINжҳҜеөҢеҘ—жҹҘиҜўдёӯжңҖз»ҸеёёдҪҝз”Ёзҡ„и°“иҜҚ

е…ҲеҲҶеёғжқҘе®ҢжҲҗжӯӨжҹҘиҜў,з„¶еҗҺеҶҚжһ„йҖ еөҢеҘ—жҹҘиҜў

е°Ҷз¬¬дёҖжӯҘжҹҘиҜўеөҢе…ҘеҲ°з¬¬дәҢжӯҘжҹҘиҜўзҡ„жқЎд»¶дёӯ,жһ„йҖ еөҢеҘ—жҹҘиҜўеҰӮдёӢ:

жң¬дҫӢдёӯ,еӯҗжҹҘиҜўзҡ„жҹҘиҜўжқЎд»¶дёҚдҫқиө–дәҺзҲ¶жҹҘиҜў,з§°дёәдёҚзӣёе…іеӯҗжҹҘиҜў,дёҖз§ҚжұӮи§Јж–№жі•жҳҜз”ұйҮҢеҗ‘еӨ–еӨ„зҗҶ,еҚіе…Ҳжү§иЎҢеӯҗжҹҘиҜў,еӯҗжҹҘиҜўзҡ„з»“жһңз”ЁдәҺе»әз«Ӣе…¶зҲ¶жҹҘиҜўзҡ„жҹҘжүҫжқЎд»¶,еҫ—еҲ°еҰӮдёӢзҡ„иҜӯеҸҘ:

з„¶еҗҺжү§иЎҢиҜҘиҜӯеҸҘ

жң¬дҫӢдёӯзҡ„жҹҘиҜўд№ҹеҸҜд»Ҙз”ЁиҮӘиә«иҝһжҺҘжқҘе®ҢжҲҗ:

еҸҜи§Ғ,е®һзҺ°еҗҢдёҖдёӘжҹҘиҜўиҜ·жұӮеҸҜд»ҘжңүеӨҡз§Қж–№жі•,еҪ“з„¶дёҚеҗҢзҡ„ж–№жі•е…¶жү§иЎҢж•ҲзҺҮеҸҜиғҪдјҡжңүе·®еҲ«,з”ҡиҮідјҡе·®еҲ«еҫҲеӨ§,иҝҷе°ұжҳҜж•°жҚ®еә“зј–зЁӢдәәе‘ҳеә”иҜҘжҺҢжҸЎзҡ„ж•°жҚ®еә“жҖ§иғҪи°ғдјҳжҠҖжңҜ

**[дҫӢ3.56]**жҹҘиҜўйҖүдҝ®дәҶиҜҫзЁӢеҗҚдёә"дҝЎжҒҜзі»з»ҹ"зҡ„еӯҰз”ҹеӯҰеҸ·е’Ңе§“еҗҚ

жң¬жҹҘиҜўж¶үеҸҠеӯҰеҸ·,е§“еҗҚе’ҢиҜҫзЁӢеҗҚдёүдёӘеұһжҖ§,еӯҰеҸ·е’Ңе§“еҗҚеӯҳж”ҫеңЁStudentиЎЁдёӯ,иҜҫзЁӢеҗҚеӯҳж”ҫеңЁCourseиЎЁдёӯ,дҪҶStudentдёҺCourseдёӨдёӘиЎЁд№Ӣй—ҙжІЎжңүзӣҙжҺҘиҒ”зі»,еҝ…йЎ»йҖҡиҝҮSCиЎЁе»әз«Ӣе®ғд»¬дәҢиҖ…д№Ӣй—ҙзҡ„иҒ”зі»,жүҖд»Ҙжң¬жҹҘиҜўе®һйҷ…дёҠж¶үеҸҠдёүдёӘе…ізі»

жң¬жҹҘиҜўеҗҢж ·еҸҜд»Ҙз”ЁиҝһжҺҘжҹҘиҜўе®һзҺ°:

жңүдәӣеөҢеҘ—жҹҘиҜўеҸҜд»Ҙз”ЁиҝһжҺҘиҝҗз®—жӣҝд»Ј,жңүдәӣжҳҜдёҚиғҪжӣҝд»Јзҡ„,жҹҘиҜўж¶үеҸҠеӨҡдёӘе…ізі»ж—¶,з”ЁеөҢеҘ—жҹҘиҜўйҖҗжӯҘжұӮи§ЈеұӮж¬Ўе…ізі»жё…жҘҡ,жҳ“дәҺжһ„йҖ ,е…·жңүз»“жһ„еҢ–зЁӢеәҸи®ҫи®Ўзҡ„дјҳзӮ№,дҪҶжҳҜзӣёжҜ”дәҺиҝһжҺҘиҝҗз®—,зӣ®еүҚе•Ҷз”Ёе…ізі»ж•°жҚ®еә“з®ЎзҗҶзі»з»ҹеҜ№еөҢеҘ—жҹҘиҜўзҡ„дјҳеҢ–еҒҡеҫ—иҝҳдёҚеӨҹе®Ңе–„,жүҖд»ҘеңЁе®һйҷ…еә”з”Ёдёӯ,иғҪеӨҹз”ЁиҝһжҺҘиҝҗз®—зҡ„жҹҘиҜўе°ҪеҸҜиғҪйҮҮз”ЁиҝһжҺҘиҝҗз®—

дҫӢ3.55е’ҢдҫӢ3.56дёӯеӯҗжҹҘиҜўзҡ„жҹҘиҜўжқЎд»¶дёҚдҫқиө–дёҺзҲ¶жҹҘиҜў,иҝҷзұ»еӯҗжҹҘиҜўз§°дёәдёҚзӣёе…іеӯҗжҹҘиҜў,дёҚзӣёе…іеӯҗжҹҘиҜўжҳҜиҫғз®ҖеҚ•зҡ„дёҖзұ»еӯҗжҹҘиҜў,еҰӮжһңеӯҗжҹҘиҜўзҡ„жҹҘиҜўжқЎд»¶дҫқиө–дәҺзҲ¶жҹҘиҜў,иҝҷзұ»еӯҗжҹҘиҜўз§°дёәзӣёе…іеӯҗжҹҘиҜў(correlated subquery),ж•ҙдёӘжҹҘиҜўиҜӯеҸҘз§°дёәзӣёе…іеөҢеҘ—жҹҘиҜў(correlated nested query)иҜӯеҸҘ

дҫӢ3.57е°ұжҳҜдёҖдёӘзӣёе…іеӯҗжҹҘиҜўзҡ„дҫӢеӯҗ

еёҰжңүжҜ”иҫғиҝҗз®—з¬Ұзҡ„еӯҗжҹҘиҜў

еёҰжңүжҜ”иҫғиҝҗз®—з¬Ұзҡ„еӯҗжҹҘиҜўжҳҜжҢҮзҲ¶жҹҘиҜўдәҺеӯҗжҹҘиҜўд№Ӣй—ҙз”ЁжҜ”иҫғиҝҗз®—з¬ҰиҝӣиЎҢиҝһжҺҘ,еҪ“з”ЁжҲ·иғҪзЎ®еҲҮзҹҘйҒ“еҶ…еұӮжҹҘиҜўиҝ”еӣһзҡ„жҳҜеҚ•дёӘеҖјж—¶,еҸҜд»Ҙз”Ё>,<.=,>=,<=,!=жҲ–<>зӯүжҜ”иҫғиҝҗз®—з¬Ұ

дҫӢеҰӮеңЁдҫӢ3.55дёӯ,з”ұдәҺдёҖдёӘеӯҰз”ҹеҸӘеҸҜиғҪеңЁдёҖдёӘзі»еӯҰд№ ,д№ҹе°ұжҳҜиҜҙеҶ…жҹҘиҜўзҡ„з»“жһңжҳҜдёҖдёӘеҖј,еӣ жӯӨеҸҜд»Ҙз”Ё=д»ЈжӣҝIN:

**[дҫӢ3.57]**жүҫеҮәжҜҸдёӘеӯҰз”ҹи¶…иҝҮд»–иҮӘе·ұйҖүдҝ®иҜҫзЁӢе№іеқҮжҲҗз»©зҡ„иҜҫзЁӢеҸ·

xжҳҜиЎЁSCзҡ„еҲ«еҗҚ,еҸҲз§°дёәе…ғз»„еҸҳйҮҸ,еҸҜд»Ҙз”ЁжқҘиЎЁзӨәSCзҡ„дёҖдёӘе…ғз»„,еҶ…еұӮжҹҘиҜўжҳҜжұӮдёҖдёӘеӯҰз”ҹжүҖжңүйҖүдҝ®иҜҫзЁӢе№іеқҮжҲҗз»©зҡ„,иҮідәҺжҳҜе“ӘдёӘеӯҰз”ҹзҡ„е№іеқҮжҲҗз»©иҰҒзңӢеҸӮж•°x.Snoзҡ„еҖј,иҖҢиҜҘеҖјжҳҜдәҺзҲ¶жҹҘиҜўзӣёе…ізҡ„,еӣ жӯӨиҝҷзұ»жҹҘиҜўз§°дёәзӣёе…іеӯҗжҹҘиҜў

иҝҷдёӘиҜӯеҸҘзҡ„дёҖз§ҚеҸҜиғҪзҡ„жү§иЎҢиҝҮзЁӢйҮҮз”Ёд»ҘдёӢдёүдёӘжӯҘйӘӨ:

  • д»ҺеӨ–еұӮжҹҘиҜўдёӯеҸ–еҮәSCзҡ„дёҖдёӘе…ғз»„x,е°Ҷе…ғз»„xзҡ„SnoеҖј(201215121)дј йҖҒз»ҷеҶ…еұӮжҹҘиҜў:

  • жү§иЎҢеҶ…еұӮ,еҫ—еҲ°88(иҝ‘дјјеҖј),з”ЁиҜҘеҖјд»ЈжӣҝеҶ…еұӮжҹҘиҜў,еҫ—еҲ°еӨ–еұӮжҹҘиҜў:

  • жү§иЎҢиҝҷдёӘжҹҘиҜў,еҫ—еҲ°

з„¶еҗҺеӨ–еұӮжҹҘиҜўеҸ–еҮәдёӢдёҖдёӘе…ғз»„йҮҚеӨҚеҒҡдёҠиҝ°1-3жӯҘйӘӨзҡ„еӨ„зҗҶ,зҹҘйҒ“еӨ–еұӮе°ұзҡ„SCе…ғз»„е…ЁйғЁеӨ„зҗҶе®ҢжҜ•,з»“жһңдёә:

жұӮи§Јзӣёе…іеӯҗжҹҘиҜўдёҚиғҪеғҸжұӮи§ЈдёҚзӣёе…іеӯҗжҹҘиҜўйӮЈж ·дёҖж¬Ўе°ҶеӯҗжҹҘиҜўжұӮи§ЈеҮәжқҘ,з„¶еҗҺжұӮи§ЈзҲ¶жҹҘиҜў,еҶ…еұӮжҹҘиҜўз”ұдәҺдёҺеӨ–еұӮжҹҘиҜўжңүе…і,еӣ жӯӨеҝ…йЎ»еҸҚеӨҚжұӮеҖј

еёҰжңүANY(SOME)жҲ–ALLи°“иҜҚзҡ„еӯҗжҹҘиҜў

еӯҗжҹҘиҜўиҝ”еӣһеҚ•еҖјж—¶еҸҜд»Ҙз”ЁжҜ”иҫғиҝҗз®—з¬Ұ,дҪҶиҝ”еӣһеӨҡеҖјж—¶иҰҒз”ЁANY(жңүзҡ„зі»з»ҹз”ЁSOME)жҲ–ALLи°“иҜҚдҝ®йҘ°з¬Ұ,иҖҢдҪҝз”ЁANYжҲ–ALLи°“иҜҚж—¶еҝ…йЎ»еҗҢж—¶дҪҝз”ЁжҜ”иҫғиҝҗз®—з¬Ұ,е…¶иҜӯд№үеҰӮдёӢжүҖзӨә:
