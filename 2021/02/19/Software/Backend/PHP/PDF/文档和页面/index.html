PHP PDF文档与页面

PDF文档由若干个页面(page)组成,每个页面包含文字或图像,本节介绍如何创建一个文档,在文档中创建页面,在页面上写入文字,并在完成时把页面发送到浏览器

提示

本章中的示例需要你的浏览器安装了最新的AdobePDF文档查看器插件,否则示例的结果将无法查看,你可以访问Adobe的网站来下载该查看器插件:http://www.adobe.com
