Spring MVC жҺ§еҲ¶еҷЁ

жң¬ж–ҮжңҖеҗҺжӣҙж–°дәҺпјҡ2021е№ҙ2жңҲ23ж—Ҙ дёӢеҚҲ

Spring MVC жҺ§еҲ¶еҷЁ

  • жҺ§еҲ¶еҷЁеӨҚжқӮжҸҗдҫӣи®ҝй—®еә”з”ЁзЁӢеәҸзҡ„иЎҢдёә,йҖҡеёёйҖҡиҝҮжҺҘеҸЈе®ҡд№үжҲ–жіЁи§Је®ҡд№үдёӨз§Қж–№жі•е®һзҺ°
  • жҺ§еҲ¶еҷЁиҙҹиҙЈи§Јжһҗз”ЁжҲ·зҡ„иҜ·жұӮе№¶е°Ҷе…¶иҪ¬жҚўдёәдёҖдёӘжЁЎеһӢ
  • еңЁSpring MVCдёӯдёҖдёӘжҺ§еҲ¶еҷЁзұ»еҸҜд»ҘеҢ…еҗ«еӨҡдёӘж–№жі•
  • еңЁSpring MVCдёӯ,еҜ№дәҺControllerзҡ„й…ҚзҪ®ж–№ејҸжңүеҫҲеӨҡз§Қ

еҲӣе»әController

е®һзҺ°ControllerжҺҘеҸЈ

  • ControllerжҳҜдёҖдёӘжҺҘеҸЈ,еңЁorg.springframework.web.servlet.mvcеҢ…дёӢ,жҺҘеҸЈдёӯеҸӘжңүдёҖдёӘж–№жі•
  • зј–еҶҷе®ҢжҜ•еҗҺ,еҺ»Springй…ҚзҪ®ж–Үд»¶дёӯжіЁеҶҢиҜ·жұӮзҡ„bean,nameеҜ№еә”иҜ·жұӮи·Ҝеҫ„,classеҜ№еә”еӨ„зҗҶиҜ·жұӮзҡ„зұ»

иҜҙжҳҺ:

  • е®һзҺ°жҺҘеҸЈControllerе®ҡд№үжҺ§еҲ¶еҷЁжҳҜиҫғиҖҒзҡ„еҠһжі•

  • зјәзӮ№жҳҜ:дёҖдёӘжҺ§еҲ¶еҷЁдёӯеҸӘжңүдёҖдёӘж–№жі•,еҰӮжһңиҰҒеӨҡдёӘж–№жі•еҲҷйңҖиҰҒе®ҡд№үеӨҡдёӘController,е®ҡд№үзҡ„ж–№ејҸжҜ”иҫғйә»зғҰ

дҪҝз”ЁжіЁи§Ј@Controller

  • @ControllerжіЁи§Јзұ»еһӢз”ЁдәҺеЈ°жҳҺSpringзұ»зҡ„е®һдҫӢжҳҜдёҖдёӘжҺ§еҲ¶еҷЁ
  • SpringеҸҜд»ҘдҪҝз”Ёжү«жҸҸжңәеҲ¶жқҘжүҫеҲ°еә”з”ЁзЁӢеәҸдёӯжүҖжңүеҹәдәҺжіЁи§Јзҡ„жҺ§еҲ¶еҷЁзұ»,дёәдәҶдҝқиҜҒSpringиғҪжүҫеҲ°дҪ зҡ„жҺ§еҲ¶еҷЁ,йңҖиҰҒеңЁй…ҚзҪ®ж–Үд»¶дёӯеЈ°жҳҺз»„д»¶жү«жҸҸ
  • еўһеҠ дёҖдёӘControllerTest2зұ»,дҪҝз”ЁжіЁи§Је®һзҺ°

  • з”ЁдәҺзұ»дёҠ,иЎЁзӨәзұ»дёӯзҡ„жүҖжңүе“Қеә”иҜ·жұӮзҡ„ж–№жі•йғҪжҳҜд»ҘиҜҘең°еқҖдҪңдёәзҲ¶и·Ҝеҫ„
  • и®ҝй—®и·Ҝеҫ„:http://localhost:8080 / йЎ№зӣ®еҗҚ/ admin /h1 , йңҖиҰҒе…ҲжҢҮе®ҡзұ»зҡ„и·Ҝеҫ„еҶҚжҢҮе®ҡж–№жі•зҡ„и·Ҝеҫ„

  • @RequestMapping()жӢ¬еҸ·дёӯй»ҳи®Өдёәvalue

  • жіЁж„Ҹ:жүҖжңүзҡ„ең°еқҖж ҸиҜ·жұӮй»ҳи®ӨйғҪдјҡжҳҜ HTTP GET зұ»еһӢзҡ„
  • ж–№жі•зә§еҲ«зҡ„жіЁи§ЈеҸҳдҪ“жңүеҰӮдёӢеҮ дёӘ:

  • е®ғжүҖжү®жј”зҡ„жҳҜ@RequestMapping(method =RequestMethod.GET)зҡ„дёҖдёӘеҝ«жҚ·ж–№ејҸ

@RequestParam

  • еҸҜжҺҘж”¶urlдёӯзҡ„й”®еҖјеҜ№(GET)
  • еҸҜжҺҘеҸ—ContentTypeжҢҮе®ҡдёәapplication/x-www-form-urlencodedзҡ„иҜ·жұӮдҪ“(POST)
  • еҰӮжһңиҜ·жұӮдёәд»ҘдёҠдёӨз§Қж–№жі•иҜ·жұӮдё”еҸӮж•°еҗҚдёҺеҪўејҸеҸӮж•°еҗҚзӣёеҗҢ,еҸҜд»ҘзңҒз•ҘиҜҘжіЁйҮҠ
  • иҝҷз§Қжғ…еҶөдёӢ,з”ЁеҲ°зҡ„еҸӮж•°еӨ„зҗҶеҷЁжҳҜRequestParamMapMethodArgumentResolver
  • жіЁж„Ҹ:еҪўејҸеҸӮж•°еҸҜд»Ҙдёәе®һдҪ“зұ»зҡ„еҜ№иұЎ,иҝҷж—¶иҜ·жұӮзҡ„й”®еҖјеҜ№дјҡиҮӘеҠЁжҢүз…§й”®еҗҚдёҺеұһжҖ§еҗҚеҢ№й…Қзҡ„еҺҹеҲҷиҮӘеҠЁжіЁе…Ҙ,иҝҷз§Қжғ…еҶөдёӢ,дёҚиғҪдҪҝз”Ёжң¬жіЁи§Ј,еҸӮж•°еӨ„зҗҶеҷЁдёәServletModelAttributeMethodProcessor,дё»иҰҒжҳҜжҠҠHttpServletRequestдёӯзҡ„иЎЁеҚ•еҸӮж•°е°ҒиЈ…еҲ°MutablePropertyValuesе®һдҫӢдёӯ,еҶҚйҖҡиҝҮеҸӮж•°зұ»еһӢе®һдҫӢеҢ–(йҖҡиҝҮжһ„йҖ еҸҚе°„еҲӣе»әUserе®һдҫӢ),еҸҚе°„еҢ№й…ҚеұһжҖ§иҝӣиЎҢеҖјзҡ„еЎ«е……

  • йҖҡиҝҮResponseBodyжіЁи§Ј,еҸҜд»Ҙиҝ”еӣһjsonж јејҸзҡ„ж•°жҚ®
