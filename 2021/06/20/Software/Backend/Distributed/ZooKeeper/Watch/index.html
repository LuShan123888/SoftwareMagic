ZooKeeper Watch

  • zookeeper 的 watcher 机制,可以分为四个过程:
    • 客户端注册 watcher
    • 服务端处理 watcher
    • 服务端触发 watcher 事件
    • 客户端回调 watcher
  • 其中客户端注册 watcher 有三种方式,调用客户端 API 可以分别通过 getData,exists,getChildren 实现,新建 WatcherDemo 类,以 exists 方法举例说明其原理

客户端发送事件通知请求
