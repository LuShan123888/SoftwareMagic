Spring Boot AOP

通知注解

  • 前置通知（@Before）：在目标方法调用之前调用通知
  • 后置通知（@After）：在目标方法完成之后调用通知
  • 环绕通知（@Around）：在被通知的方法调用之前和调用之后执行自定义的方法
  • 返回通知（@AfterReturning）：在目标方法成功执行之后调用通知
  • 异常通知（@AfterThrowing）：在目标方法抛出异常之后调用通知
