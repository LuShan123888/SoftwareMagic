设计模式 原型模式

  • 通过 new 产生一个对象需要非常繁琐的数据准备或访问权限，则可以使用原型模式
  • 原型模式就是构建一个对象的原型，通过原型来创建出新的对象

浅克隆

  • 特点:如果某个对象的属性是其他对象时,则克隆出来的新对象中的对应属性是引用原对象的

深克隆

  • 主要通过重写 clone 方法实现深克隆
