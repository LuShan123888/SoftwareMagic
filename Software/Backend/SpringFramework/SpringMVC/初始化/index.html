  • / 和 /* 的区别:< url-pattern > / </ url-pattern > 不会匹配到.jsp, 只针对我们编写的请求;即:.jsp 不会进入Spring的 DispatcherServlet类 ,< url-pattern > /* </ url-pattern> 会匹配 *.jsp,会出现返回 jsp视图 时再次进入Spring的DispatcherServlet 类,导致找不到对应的controller而所以报404错
