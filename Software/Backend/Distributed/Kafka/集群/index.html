创建Topic

查看Topic

  • 运行describe topics命令
  • leader:负责给定Partition的所有读写的节点,每个节点都可能成为Partition随机选择的leader
  • replicas:复制此Partition日志的节点列表,无论它们是leader还是当前处于存活状态
  • isr:是一组 “in-sync” replicas,这是replicas列表的一个子集,它当前处于存活状态,并补充leader
