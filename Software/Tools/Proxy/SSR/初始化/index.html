ShadowSocksR 初始化

ShadowSocksR 初始化

安装

  1. 下载并执行安装脚本
  1. 输入对应的参数 执行完上述的脚本代码后,会进入到输入参数的界面,包括服务器端口,密码,加密方式,协议,混淆,可以直接输入回车选择默认值,也可以输入相应的值选择对应的选项
  2. 建议加密方式:aes-256-cfb
  3. 建议协议插件:auth_shal_v4
  4. 建议混合插件:plain
  5. 全部选择结束后,会看到如下界面,就说明搭建ssr成功了
