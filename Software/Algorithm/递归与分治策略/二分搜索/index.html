二分搜索

算法分析

  • 将n个元素分成个数大致相同的两半, 取a[n/2]与x作比较
  • 如果x=a[n/2], 则找到x, 算法终止, 如果x<a[n/2], 则只在数组a的左半部继续搜索x, 如果x>a[n/2], 则只在数组a的右半部分继续搜索x

代码实现

  • BinarySearch(T[] a, T x, int n):在有序数组a[n]的中查找x, 如果找到返回x的数组下标, 如果未找到则返回-1
