问题描述

  • 将 sourceX, sourceY 转换成 targetX, targetY 最少需要多少次计算，给定两个数字（x, y)，允许以下两种计算：
    • 1.同时对两个数加1，即（x, y) -> (x+1, y+1)
    • 2.同时对两个数乘2，即（x, y) -> (x2, y2)
  • 求要将（x, y）转换成（X,Y)，至少需要多次计算，如果不能转换，返回-1
