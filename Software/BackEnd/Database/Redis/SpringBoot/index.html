Redis Spring Boot整合

pom.xml

  • 在Spring Boot2.x之后,原来使用的jedis被替换为lettuce
  • jedis使用直连server,多线程操作时有安全隐患,可使用jedis pool连接池改善,类似BIO模式
  • lettuce底层使用netty实例可以在多个线程中进行共享,不存在线程不安全的情况,可以减少线程数据,类似NIO模式
