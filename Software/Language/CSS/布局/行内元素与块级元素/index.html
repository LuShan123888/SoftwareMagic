CSS еқ—зә§е…ғзҙ дёҺиЎҢеҶ…е…ғзҙ 

еқ—зә§е…ғзҙ (block):зӢ¬еҚ дёҖиЎҢ,еҜ№е®Ҫй«ҳзҡ„еұһжҖ§еҖјз”ҹж•Ҳ;еҰӮжһңдёҚз»ҷе®ҪеәҰ,еқ—зә§е…ғзҙ е°ұй»ҳи®ӨдёәжөҸи§ҲеҷЁзҡ„е®ҪеәҰ,еҚіе°ұжҳҜ100%е®Ҫ

иЎҢеҶ…е…ғзҙ (inline):еҸҜд»ҘеӨҡдёӘж ҮзӯҫеӯҳеңЁдёҖиЎҢ,еҜ№е®Ҫй«ҳеұһжҖ§еҖјдёҚз”ҹж•Ҳ,е®Ңе…Ёйқ еҶ…е®№ж’‘ејҖе®Ҫй«ҳ

иЎҢеҶ…еқ—е…ғзҙ (inline-block):з»“еҗҲзҡ„иЎҢеҶ…е’Ңеқ—зә§зҡ„дјҳзӮ№,ж—ўеҸҜд»Ҙи®ҫзҪ®й•ҝе®Ҫ,еҸҜд»Ҙи®©paddingе’Ңmarginз”ҹж•Ҳ,еҸҲеҸҜд»Ҙе’Ңе…¶д»–иЎҢеҶ…е…ғзҙ е№¶жҺ’

еҪ“жҠҠиЎҢеҶ…е…ғзҙ и®ҫзҪ®е®Ңfloat:left/rightеҗҺ,иҜҘиЎҢеҶ…е…ғзҙ зҡ„displayеұһжҖ§дјҡиў«иөӢдәҲblockеҖј,дё”жӢҘжңүжө®еҠЁзү№жҖ§,иЎҢеҶ…е…ғзҙ еҺ»йҷӨдәҶд№Ӣй—ҙзҡ„иҺ«еҗҚз©әзҷҪ

  • text-alignеұһжҖ§еҜ№еқ—зә§е…ғзҙ иө·дҪңз”Ё,еҜ№иЎҢеҶ…е…ғзҙ дёҚиө·дҪңз”Ё:еҺҹеӣ жҳҜеқ—зә§ж ҮзӯҫеҰӮжһңдёҚз»ҷе®ҪеәҰ,еқ—зә§е…ғзҙ е°ұй»ҳи®ӨдёәжөҸи§ҲеҷЁзҡ„е®ҪеәҰ,еҚіе°ұжҳҜ100%е®Ҫ,йӮЈд№ҲеңЁ100%зҡ„е®ҪеәҰдёӯеұ…дёӯз”ҹж•Ҳ;дҪҶжҳҜиЎҢеҶ…е…ғзҙ зҡ„е®Ҫе®Ңе…ЁжҳҜйқ еҶ…е®№ж’‘ејҖ,жүҖд»Ҙе®ҪеәҰе°ұжҳҜеҶ…е®№ж’‘ејҖзҡ„е®Ҫ:

  • еқ—зә§е…ғзҙ еҸҜд»Ҙи®ҫзҪ®weithе’Ңheight,иЎҢеҶ…е…ғзҙ и®ҫзҪ®widthе’Ңheightж— ж•Ҳ,иҖҢдё”еқ—зә§е…ғзҙ еҚідҪҝи®ҫзҪ®е®ҪеәҰд№ҹиҝҳжҳҜзӢ¬еҚ дёҖиЎҢ,жіЁж„ҸдҪҶеқ—зә§е…ғзҙ еҪ“жІЎжңүжҳҺзЎ®жҢҮе®ҡ width е’Ң height еҖјж—¶,еқ—зә§е…ғзҙ е°әеҜёз”ұеҶ…е®№зЎ®е®ҡ,еҪ“жҢҮе®ҡдәҶ width е’Ң height зҡ„еҖјж—¶,еҶ…е®№и¶…еҮәеқ—зә§е…ғзҙ зҡ„е°әеҜёе°ұдјҡжәўеҮә,иҝҷж—¶еқ—зә§е…ғзҙ иҰҒе‘ҲзҺ°д»Җд№ҲиЎҢдёәиҰҒзңӢе…¶ overflow зҡ„еҖј(visible,hidden,overflow,scroll)

  • иЎҢеҶ…е…ғзҙ зҡ„padding-topе’Ңpadding-bottomдјҡиө·дҪңз”Ё,дёҚиҝҮе°ұеғҸи„ұзҰ»дәҶж ҮеҮҶжөҒдёҖж ·(еҚіpadding-leftе’Ңpadding-rightзҡ„еұӮзә§жҜ”е…¶д»–е…ғзҙ й«ҳ),е№¶дёҚдјҡеҚ жҚ®дҪҚзҪ®,е№¶дё”иҝҳжҠҠе…¶д»–е…ғзҙ з»ҷзӣ–дҪҸдәҶ,дҪҶжҳҜ,еҒҮеҰӮinlineзҡ„е…ғзҙ жІЎжңүеҶ…е®№,"padding-top,padding-bottom"е°ҶдёҚиө·дҪңз”Ё,еҰӮжһңжғіиҰҒиө·дҪңз”Ё,еҸӘйңҖиҰҒз»ҷpadding-leftжҲ–иҖ…padding-rightи®ҫзҪ®дёҖдёӘеҖј,жҲ–иҖ…еҪ“inlineзҡ„е…ғзҙ жңүеҶ…е®№ж—¶е°ұдјҡиө·дҪңз”Ё
