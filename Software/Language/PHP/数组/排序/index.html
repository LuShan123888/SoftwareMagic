PHP 数组排序

  • 排序(sorting)改变了数组中元素的键名来反映新顺序，例如，可以使用排序来按从大到小排列一个得分清单，按字母表排列一个名字清单或根据用户提交的消息多少来排列一组用户
  • PHP提供三种对数组排序的方法:按键排序，不改变键的按值排序，按值排序并改变键，每种排序方式都可以按升序，降序或用户定义函数的顺序排序
