比较运算符

  • 比较运算符在逻辑语句中使用,以测定变量或值是否相等
  • x=5,下面的表格解释了比较运算符:
运算符 描述 比较 返回值
== 等于 x==8 false
x==5 true
=== 绝对等于(值和类型均相等) x===“5” false
x===5 ** true
!= 不等于 x!=8 true
!== 不绝对等于(值和类型有一个不相等,或两个都不相等) x!==“5” true
x!==5 ** false
> 大于 x>8 false
< 小于 x<8 true
>= 大于或等于 x>=8 false
<= 小于或等于 x<=8 true
