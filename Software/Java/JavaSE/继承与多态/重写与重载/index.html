Java йҮҚеҶҷдёҺйҮҚиҪҪ

ж–№жі•зҡ„йҮҚиҪҪ

  • ж–№жі•йҮҚиҪҪе°ұжҳҜеҗҢдёҖзұ»дёӯеӯҳеңЁеӨҡдёӘж–№жі•еҗҚзӣёеҗҢдҪҶеҸӮж•°дёҚеҗҢзҡ„ж–№жі•,еҸӮж•°зҡ„е·®ејӮеҢ…жӢ¬еҪўејҸеҸӮж•°зҡ„дёӘж•°,зұ»еһӢзӯү,еңЁдҫӢ6-2дёӯ,зұ»Aе®ҡд№үдәҶ3дёӘtest()ж–№жі•,е®ғд»¬зҡ„еҸӮж•°зұ»еһӢдёҚеҗҢ
  • ж–№жі•и°ғз”Ёзҡ„еҢ№й…ҚеӨ„зҗҶеҺҹеҲҷжҳҜ:йҰ–е…ҲжҢү"зІҫзЎ®еҢ№й…Қ"еҺҹеҲҷеҺ»жҹҘжүҫеҢ№й…Қж–№жі•,еҰӮжһңжүҫдёҚеҲ°,еҲҷжҢү"иҮӘеҠЁзұ»еһӢиҪ¬жҚўеҢ№й…Қ"еҺҹеҲҷеҺ»жҹҘжүҫиғҪеҢ№й…Қзҡ„ж–№жі•
  • жүҖи°“"зІҫзЎ®еҢ№й…Қ"е°ұжҳҜе®һеҸӮе’ҢеҪўеҸӮзұ»еһӢе®Ңе…ЁдёҖиҮҙ,жүҖи°“"иҮӘеҠЁзұ»еһӢиҪ¬жҚўеҢ№й…Қ"жҳҜжҢҮиҷҪз„¶е®һеҸӮе’ҢеҪўеҸӮзұ»еһӢдёҚеҗҢ,дҪҶиғҪе°Ҷе®һеҸӮзҡ„ж•°жҚ®жҢүиҮӘеҠЁиҪ¬жҚўеҺҹеҲҷиөӢеҖјз»ҷеҪўеҸӮ

[дҫӢ6-2]:ж–№жі•и°ғз”Ёзҡ„еҢ№й…ҚжөӢиҜ•

  • ж №жҚ®ж–№жі•и°ғз”Ёзҡ„еҢ№й…ҚеҺҹеҲҷ,дёҚйҡҫеҸ‘зҺ°иҝҗиЎҢзЁӢеәҸж—¶е°Ҷеҫ—еҲ°еҰӮдёӢз»“жһң:
  • еҰӮжһңе°Ҷд»ҘдёҠtest(int x)ж–№жі•жіЁйҮҠжҺү,з»“жһңдёә:
  • иҜҙжҳҺ:еӣ дёәе®һеҸӮ5й»ҳи®ӨдёәintеһӢж•°жҚ®,еӣ жӯӨ,жңүtest(int x)ж–№жі•еӯҳеңЁж—¶е°ҶжҢү"зІҫзЎ®еҢ№й…Қ"еҺҹеҲҷеӨ„зҗҶ,дҪҶеҰӮжһңж— иҜҘж–№жі•еӯҳеңЁ,е°ҶжҢү"иҪ¬жҚўеҢ№й…Қ"еҺҹеҲҷдјҳе…ҲиҖғиҷ‘еҢ№й…Қtest(long x)ж–№жі•

[дҫӢ6-3]:д»ҺеӨҚж•°ж–№жі•зҗҶи§ЈеӨҡжҖҒжҖ§

  • д»ҘдёҠжңү3дёӘж–№жі•е®һзҺ°еӨҚж•°зҡ„зӣёеҠ иҝҗз®—,е…¶дёӯжңүдёӨдёӘдёәе®һдҫӢж–№жі•,дёҖдёӘж—¶йқҷжҖҒж–№жі•,е®ғд»¬зҡ„еҸӮж•°еҪўејҸжҳҜдёҚеҗҢзҡ„,и°ғз”Ёж–№жі•ж—¶е°Ҷж №жҚ®еҸӮж•°еҪўжҖҒеҶіе®ҡеҢ№й…Қе“ӘдёӘж–№жі•,жіЁж„ҸйқҷжҖҒж–№жі•е’Ңе®һдҫӢж–№жі•зҡ„и°ғз”Ёе·®ејӮ,е®һдҫӢж–№жі•дёҖе®ҡиҰҒжңүдёҖдёӘеҜ№иұЎдҪңдёәеүҚзјҖ:йқҷжҖҒж–№жі•еҲҷдёҚдҫқиө–еҜ№иұЎ
  • д»ҘдёҠ3дёӘж–№жі•иҝӣиЎҢеӨҚж•°зӣёеҠ ж—¶е°Ҷдә§з”ҹдёҖдёӘж–°зҡ„еӨҚж•°дҪңдёәж–№жі•зҡ„иҝ”еӣһеҖј,е№¶дёҚж”№еҸҳеҸӮдёҺиҝҗз®—зҡ„дёӨдёӘеӨҚж•°еҜ№иұЎзҡ„еҖј,еҰӮжһңиҰҒж”№еҸҳи°ғз”Ёж–№жі•зҡ„еӨҚж•°зҡ„еҖј,еҲҷж–№жі•и®ҫи®ЎдёәеҰӮдёӢеҪўејҸ:

ж–№жі•зҡ„иҰҶзӣ–дёҺйҡҗи—Ҹ

  • еӯҗзұ»е°Ҷз»§жүҝзҲ¶зұ»зҡ„йқһз§Ғжңүж–№жі•,еңЁеӯҗзұ»д№ҹеҸҜд»ҘеҜ№зҲ¶зұ»е®ҡд№үзҡ„ж–№жі•йҮҚж–°е®ҡд№ү,иҝҷж—¶е°Ҷдә§з”ҹж–№жі•иҰҶзӣ–,д№ҹе°ұжҳҜйҖҡиҝҮеӯҗзұ»еҜ№иұЎи®ҝй—®зҡ„ж–№жі•жҳҜеӯҗзұ»иҮӘе·ұйҮҚж–°е®ҡд№үзҡ„ж–№жі•,йңҖиҰҒжіЁж„Ҹзҡ„жҳҜ,еӯҗзұ»еңЁйҮҚж–°е®ҡд№үзҲ¶зұ»е·Іжңүзҡ„ж–№жі•ж—¶,еә”дҝқжҢҒдёҺзҲ¶зұ»е®Ңе…ЁзӣёеҗҢзҡ„ж–№жі•еӨҙйғЁеЈ°жҳҺ,еҚіеә”дёҺзҲ¶зұ»е…·жңүе®Ңе…ЁзӣёеҗҢзҡ„ж–№жі•еҗҚ,еҸӮж•°еҲ—иЎЁ,иҝ”еӣһзұ»еһӢдёҖиҲ¬д№ҹиҰҒзӣёеҗҢ
  • дҫӢеҰӮ,еңЁд»ҘдёӢзұ»Bдёӯе®ҡд№үзҡ„ж–№жі•,еҸӘжңүtest(int x)еӯҳеңЁеҜ№дҫӢ6-2дёӯзұ»Aзҡ„ж–№жі•иҰҶзӣ–:
  • зҲ¶зұ»зҡ„е®һдҫӢж–№жі•иў«еӯҗзұ»зҡ„еҗҢеҗҚе®һдҫӢж–№жі•иҰҶзӣ–,зҲ¶зұ»зҡ„йқҷжҖҒж–№жі•иў«еӯҗзұ»зҡ„еҗҢеҗҚйқҷжҖҒж–№жі•йҡҗи—Ҹ,зҲ¶зұ»зҡ„е®һдҫӢеҸҳйҮҸе’Ңзұ»еҸҳйҮҸеҸҜд»Ҙиў«еӯҗзұ»зҡ„е®һдҫӢеҸҳйҮҸе’Ңзұ»еҸҳйҮҸйҡҗи—Ҹ
  • йҖҡиҝҮзҲ¶зұ»еј•з”ЁеҸҜд»ҘжҡҙйңІйҡҗи—Ҹзҡ„еҸҳйҮҸе’Ңж–№жі•
  • ж–№жі•иҰҶзӣ–дёҚиғҪж”№еҸҳж–№жі•зҡ„йқҷжҖҒдёҺйқһйқҷжҖҒеұһжҖ§,еӯҗзұ»дёӯдёҚиғҪе°ҶзҲ¶зұ»зҡ„е®һдҫӢж–№жі•е®ҡд№үдёәйқҷжҖҒж–№жі•,д№ҹдёҚиғҪе°ҶзҲ¶зұ»зҡ„йқҷжҖҒж–№жі•е®ҡд№үдёәе®һдҫӢж–№жі•
  • дёҚе…Ғи®ёеӯҗзұ»дёӯж–№жі•зҡ„и®ҝй—®дҝ®йҘ°з¬ҰжҜ”зҲ¶зұ»жңүжӣҙеӨҡзҡ„йҷҗеҲ¶,дҫӢеҰӮ,дёҚиғҪе°ҶзҲ¶зұ»е®ҡд№үдёӯз”Ёpublicдҝ®йҘ°зҡ„ж–№жі•еңЁеӯҗзұ»дёӯйҮҚе®ҡд№үдёәprivateж–№жі•,дҪҶеҸҜд»Ҙе°ҶзҲ¶зұ»зҡ„privateж–№жі•йҮҚе®ҡд№үдёәpublicж–№жі•,йҖҡеёёеә”е°Ҷеӯҗзұ»дёӯж–№жі•и®ҝй—®дҝ®йҘ°дёҺзҲ¶зұ»дёӯзҡ„дҝқжҢҒдёҖиҮҙ

зҲ¶зұ»еј•з”ЁжҢҮеҗ‘еӯҗзұ»еҜ№иұЎ

  • еҰӮжһңеӯҗзұ»дёӯе®ҡд№үдәҶдёҺзҲ¶зұ»еҗҢеҗҚзҡ„еұһжҖ§,еңЁеӯҗзұ»дёӯе°Ҷйҡҗи—ҸжқҘиҮӘзҲ¶зұ»зҡ„еҗҢеҗҚеұһжҖ§еҸҳйҮҸ,иҝҷйҮҢд№ҹжҳҜ"жңҖиҝ‘дјҳе…ҲеҺҹеҲҷ",иҮӘе·ұзұ»дёӯжңүе°ұдёҚдјҡеҺ»жүҫзҲ¶зұ»зҡ„

иҜҙжҳҺ

  • жҜҸдёӘExtendShowзұ»еһӢзҡ„еҜ№иұЎе°ҶжӢҘжңү4дёӘеұһжҖ§,е…¶дёӯжңүдёӨдёӘyеұһжҖ§:дёҖдёӘжҳҜеӯҗзұ»е®ҡд№үзҡ„,дёҖдёӘжҳҜзҲ¶зұ»е®ҡд№үзҡ„,еңЁеӯҗзұ»дёӯ,жҹҘжүҫжҲҗе‘ҳеұһжҖ§ж—¶е°Ҷдјҳе…ҲеҢ№й…Қжң¬зұ»е®ҡд№үзҡ„еұһжҖ§,еңЁеӯҗзұ»дёӯжІЎжңүжүҫеҲ°зҡ„еұһжҖ§жүҚдјҡеҲ°зҲ¶зұ»дёӯеҺ»жҹҘжүҫ,д№ҹе°ұжҳҜеңЁеӯҗзұ»дёӯе°Ҷйҡҗи—ҸзҲ¶зұ»еҲқе§ӢеҢ–иҝҮзҡ„еҗҢеҗҚеұһжҖ§
  • е°Ҷеӯҗзұ»еҜ№иұЎиөӢеҖјз»ҷзҲ¶зұ»зҡ„еј•з”ЁеҸҳйҮҸеҗҺ,йҖҡиҝҮиҜҘеј•з”ЁеҸҳйҮҸеҺ»и®ҝй—®еҜ№иұЎзҡ„жҲҗе‘ҳж—¶,и®ҝй—®зҡ„иЎҢдёәж–№жі•жҳҜеӯҗзұ»еҜ№иұЎзҡ„иҰҶзӣ–ж–№жі•,иҖҢеұһжҖ§еҖјеҚҙжҳҜзҲ¶зұ»зҡ„,еҺҹеӣ еңЁдәҺеҜ№иұЎжү§иЎҢж–№жі•ж—¶з”ұе®һйҷ…еҜ№иұЎзҡ„зұ»еһӢеҶіе®ҡ,иҖҢдёҚжҳҜеј•з”ЁеҸҳйҮҸзұ»еһӢ,и®ҝй—®еұһжҖ§ж—¶еҲҷз”ұеј•з”ЁеҸҳйҮҸзҡ„зұ»еһӢеҶіе®ҡ,еӣ дёәзј–иҜ‘зЁӢеәҸеңЁеҲҶжһҗзЁӢеәҸж—¶жҳҜеҹәдәҺзұ»еһӢжқҘи®ҝй—®е“ӘдёӘеұһжҖ§еҸҳйҮҸ
  • йқҷжҖҒжҲҗе‘ҳжҳҜдҫқиө–зұ»зҡ„,йқҷжҖҒжҲҗе‘ҳзҡ„и®ҝй—®еҹәдәҺеј•з”ЁеҸҳйҮҸе’Ңзҡ„зұ»еһӢ,иҖҢдёҚжҳҜеҜ№иұЎзұ»еһӢ,е°Ҫз®ЎзҲ¶зұ»еј•з”ЁеҸҳйҮҸеҸҜжҢҮеҗ‘еӯҗзұ»еҜ№иұЎ,дҪҶйҖҡиҝҮзҲ¶зұ»еј•з”ЁеҸҳйҮҸи®ҝй—®зҡ„йқҷжҖҒж–№жі•е’ҢйқҷжҖҒж–№жі•еқҮжҳҜзҲ¶зұ»зҡ„
  • з»ҶеҝғдҪ“дјҡйҡҗи—Ҹе’ҢиҰҶзӣ–еңЁз”ЁиҜҚдёҠзҡ„е·®ејӮ,еҪ“еӯҗзұ»дёҺзҲ¶зұ»жңүзӣёеҗҢжҲҗе‘ҳж—¶,йҖҡиҝҮеӯҗзұ»еј•з”Ёи®ҝй—®зҡ„жҲҗе‘ҳеқҮжҳҜеӯҗзұ»е®ҡд№үзҡ„,й—®йўҳеңЁдәҺйҖҡиҝҮзҲ¶зұ»еј•з”Ёж“ҚдҪңдёҖдёӘеӯҗзұ»еҜ№иұЎж—¶е°ұжңүе·®ејӮдәҶ,йҡҗи—Ҹзҡ„зҲ¶зұ»жҲҗе‘ҳеҶҚзҺ°,иҖҢиҰҶзӣ–зҡ„жҲҗе‘ҳиў«еӯҗзұ»"жӣҝд»Ј",еҸӘжңүе®һдҫӢж–№жі•дјҡжҳҜеӯҗзұ»е®ҡд№үзҡ„,еҜ№иұЎеұһжҖ§,йқҷжҖҒеұһжҖ§,йқҷжҖҒж–№жі•еқҮжҳҜжҢҮзҲ¶зұ»е®ҡд№үзҡ„

еҜ№иұЎеј•з”ЁиҪ¬жҚў

еҜ№иұЎеј•з”ЁиөӢеҖјеҢ№й…Қ

  • е…Ғи®ёе°Ҷеӯҗзұ»еҜ№иұЎиөӢеҖјз»ҷзҲ¶зұ»еј•з”Ё,иҝҷз§ҚиөӢеҖјд№ҹз»ҸеёёеҸ‘з”ҹеңЁж–№жі•и°ғз”Ёзҡ„еҸӮж•°дј йҖ’ж—¶,еҰӮжһңдёҖдёӘж–№жі•зҡ„еҪўејҸеҸӮж•°е®ҡд№үзҡ„жҳҜзҲ¶зұ»еј•з”Ёзұ»еһӢ,йӮЈд№Ҳи°ғз”ЁиҝҷдёӘж–№жі•ж—¶,еҸҜд»ҘдҪҝз”Ёеӯҗзұ»еҜ№иұЎдҪңдёәе®һйҷ…еҸӮж•°,еҪ“з„¶,д»»дҪ•ж–№жі•и°ғз”Ёе°Ҷдјҳе…ҲиҖғиҷ‘еҸӮж•°зІҫзЎ®еҢ№й…Қ,з„¶еҗҺжүҚиҖғиҷ‘иҪ¬жҚўеҢ№й…Қ

[дҫӢ6-10]:ж–№жі•зҡ„еј•з”Ёзұ»еһӢеҸӮж•°еҢ№й…ҚеӨ„зҗҶ

  • еҰӮжһңе°Ҷд»ҘдёҠзҡ„test(String str)ж–№жі•е®ҡд№үжіЁйҮҠжҺү,еҲҷиҝҗиЎҢз»“жһңдёә:
  • жіЁж„Ҹ:з”ұдәҺObjectзұ»жҳҜз»§жүҝеұӮж¬ЎдёӯжңҖй«ҳеұӮзҡ„зұ»,жүҖд»Ҙд»»дҪ•еҜ№иұЎеқҮеҸҜеҢ№й…ҚObjectзұ»еһӢзҡ„еҪўеҸӮ,Javaзұ»еә“дёӯжңүдёҚе°‘ж–№жі•зҡ„еҸӮж•°жңӘObjectзұ»еһӢ,ж–№жі•е®ҡд№үиҖғиҷ‘дҪ“зҺ°дәҶйҖҡз”ЁжҖ§,еңЁJDK1.5д№ӢеҗҺз”ҡиҮіеҹәжң¬зұ»еһӢзҡ„ж•°жҚ®д№ҹеҸҜд»ҘиөӢеҖјз»ҷObjectзұ»еһӢжҲ–зӣёеә”еҢ…иЈ…зұ»еһӢзҡ„еј•з”ЁеҸҳйҮҸ,е®ғжҳҜйҖҡиҝҮе°Ҷеҹәжң¬зұ»еһӢиҮӘеҠЁеҢ…иЈ…жҲҗзӣёеә”зұ»еһӢзҡ„еҜ№иұЎжқҘе®һзҺ°иҪ¬жҚўиөӢеҖјзҡ„,дҫӢеҰӮ,intзұ»еһӢж•°жҚ®еҢ…иЈ…жҲҗIntegerзұ»еһӢзҡ„еҜ№иұЎ
