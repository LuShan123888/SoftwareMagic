Nginx 重定向

301永久重定向

  • 浏览器地址会显示跳转新的URL地址
  • 这样用户打开example.com时候就会转到www.example.com

302临时重定向

  • 浏览器地址会显示跳转新的URL地址
  • 这样用户打开example.com时候就会转到www.example.com

根据客户端重定向

  • 如果请求来自于移动端,则重定向至移动站点
  • permanent表示301永久重定向
