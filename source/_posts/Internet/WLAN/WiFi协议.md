# WiFi协议

## 802.11b

- 最高速率11Mbps
- 调制方式速率(Mbps)DSSS1DSSS2CCK5.5CCK11

| 调制方式 | 速率(Mbps) |
| -------- | ---------- |
| DSSS     | 1          |
| DSSS     | 2          |
| CCK      | 5.5        |
| CCK      | 11         |
