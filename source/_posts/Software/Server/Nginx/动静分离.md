# Nginx 动静分离

## 图片服务器

```nginx
server {
    listen 80;
    listen [::]:80;
    server_name  static.tally.ltd;
    location / {
        expires 24h;
        alias /deploy/static/;
        autoindex on; # 打开浏览功能
    }

    error_page 404 /404.html;
    location = /404.html {
    }

    error_page 500 502 503 504 /50x.html;
    location = /50x.html {
    }
}
```
