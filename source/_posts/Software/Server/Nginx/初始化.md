# Nginx 初始化

## Docker

```bash
$ docker run -d --name nginx -p 80:80 \
--hostname nginx\
-v ~/DockerVolumes/nginx1.18/conf:/etc/nginx \
-v ~/DockerVolumes/nginx1.18/html:/usr/share/nginx/html  \
nginx:1.18
```

## 安装

### Brew

```shell
$ brew install nginx
```

### CentOS

```shell
$ sudo yum -y install nginx
```
