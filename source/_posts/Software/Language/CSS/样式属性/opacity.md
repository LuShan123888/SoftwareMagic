# CSS opacity

**属性定义及使用说明**

Opacity属性设置一个元素了透明度级别。

**语法**

```
 opacity: value|inherit;
```

| 值      | 描述                                               |
| :------ | :------------------------------------------------- |
| *value* | 指定不透明度，从0.0（完全透明）到1.0（完全不透明） |
| inherit | Opacity属性的值应该从父元素继承                    |
