# CSS float&clear

## float

**属性定义及使用说明**

float属性指定一个盒子(元素)是否应该浮动

**注意**:绝对定位的元素忽略float属性!

**属性值**

| 值      | 描述                                              |
| :------ | :------------------------------------------------ |
| left    | 元素向左浮动                                      |
| right   | 元素向右浮动                                      |
| none    | 默认值,元素不浮动,并会显示在其在文本中出现的位置, |
| inherit | 规定应该从父元素继承 float 属性的值,              |

## clear

**属性定义及使用说明**

clear属性指定段落的左侧或右侧不允许浮动的元素

**属性值**

| 值      | 描述                                  |
| :------ | :------------------------------------ |
| left    | 在左侧不允许浮动元素,                |
| right   | 在右侧不允许浮动元素,                |
| both    | 在左右两侧均不允许浮动元素,          |
| none    | 默认值,允许浮动元素出现在两侧,      |
| inherit | 规定应该从父元素继承 clear 属性的值, |
