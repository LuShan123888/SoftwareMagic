# Vue 问题解决

## vue-router 路由切换后滚动条未返回顶部

**解决方法**:在`vue-router`的配置文件中加入如下配置

```js
const router = new VueRouter({
    routes,
    //切换路由后,滚动条回到顶部
    scrollBehavior(to,from,saveTop){
        if(saveTop){
            return saveTop;
        }else{
            return {x:0,y:0}
        }
    }
})
```
