# Redis 初始化

## 安装

### Docker

```shell
$ docker run  -d  
-p 6379:6379 \
--name redis \
--hostname redis \
-v ~/DockerVolumes/redis/conf:/etc/redis \
-v ~/DockerVolumes/redis/data:/data \
redis:6.0.8 redis-server /etc/redis/redis.conf
```

### Homebrew

```bash
brew install redis
```
