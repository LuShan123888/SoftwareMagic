## 编译时出现OOM

**Windows**

- `%MAVEN_HOME%\bin\mvn.bat`

```properties
 @REM set MAVEN_OPTS=-Xdebug -Xnoagent -Djava.compiler=NONE...
# 添加在上一行下面
set MAVEN_OPTS=-Xmx512m -XX:MaxPermSize=128m
```

**Unix-like system**

```
export MAVEN_OPTS="-Xmx512m -XX:MaxPermSize=128m"
```
