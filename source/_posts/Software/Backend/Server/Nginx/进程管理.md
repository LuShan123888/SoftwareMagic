# Nginx 进程管理

## 启动Nginx

```bash
$ service nginx
```

## 配置文件

- 检查配置文件

```shell
$ service nginx configtest
```

- 重新载入配置文件

```shell
$ service nginx reload
```

## 重启Nginx

```shell
$ service nginx reopen
```

## 停止Nginx

```shell
$ service nginx stop
```

## 安全退出Nginx

```shell
$ service nginx quit
```
