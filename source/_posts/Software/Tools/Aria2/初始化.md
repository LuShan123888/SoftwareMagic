# Aria2 初始化

## 安装

- 下载并运行脚本

```shell
$ wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/aria2.sh
$ chmod +x aria2.sh
$ bash aria2.sh
```

## 进程管理

- 启动:service aria2 start
- 停止:service aria2 stop
- 重启:service aria2 restart
- 查看状态:service aria2 status
