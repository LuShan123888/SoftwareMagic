# SSH 初始化

## 安装ssh

### CentOS

```bash
sudo yum install sshd
```

### Ubuntu

- 安装 SSH

```bash
apt-get install openssh-server
```

- 安装 SSH 的客户端

```bash
apt-get install openssh-client
```

## 进程管理

```bash
service ssh start/stop/restart
# OR
service sshd start/stop/restart
```
