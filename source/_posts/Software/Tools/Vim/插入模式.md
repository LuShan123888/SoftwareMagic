# Vim 插入模式（Insert-mode)

## 进入插入模式

- 在编辑模式下正确定位光标之后，可用以下命令切换到插入模式：
    - i：在光标左侧插入正文。
    - a：在光标右侧插入正文。
    - o：在光标所在行的下一行增添新行。
    - O：在光标所在行的上一行增添新行。
    - I：在光标所在行的开头插入。
    - A：在光标所在行的末尾插入。

## 退出插入模式

- 退出插入模式的方法是，按 ESC 键或组合键 Ctrl+[ ，退出插入模式之后，将会进入编辑模式。
